# Adityathebe Tap

A Homebrew tap for my macOS menubar apps.

## Available Apps

| App                                                                       | Description                                          |
| ------------------------------------------------------------------------- | ---------------------------------------------------- |
| [DayProgressMenubar](https://github.com/adityathebe/day-progress-menubar) | Menubar app showing progress through the current day |

## Installation

First, tap this repository:

```bash
brew tap adityathebe/tap
```

Then install:

```bash
brew install --cask day-progress-menubar
```

Or install directly without tapping first:

```bash
brew install --cask adityathebe/tap/day-progress-menubar
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
